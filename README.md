# NSIPT
# Information:
- A small NetSec tool used for IP related scanning.



# Installation:
(Linux)
- $ git clone https://github.com/DoxingMethod/NSIPT.git
- $ cd NSIPT
- $ chmod +x IPT
- $ ./IPT

(Windows)
- $ git clone https://github.com/DoxingMethod/NSIPT.git
- $ cd NSIPT
- (Rename ITP to IPT.py)
- $ python IPT.py)
